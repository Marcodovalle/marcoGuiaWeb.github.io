Guia de Propriedades e Funções do JavaScript

1. Tipos de Dados

  • Number
    • Number.isNaN(): Verifica se o valor é NaN (Not-a-Number).
    • Number.isInteger(): Verifica se o valor é um número inteiro.
    • Number.parseFloat(): Converte uma string em um número de ponto flutuante.
    • Number.parseInt(): Converte uma string em um número inteiro.
  • String
    • string.length: Retorna o comprimento da string.
    • string.charAt(index): Retorna o caractere na posição especificada.
    • string.includes(substring): Verifica se a string contém a substring especificada.
    • string.startsWith(substring): Verifica se a string começa com a substring especificada.
    • string.endsWith(substring): Verifica se a string termina com a substring especificada.
    • string.indexOf(substring): Retorna o índice da primeira ocorrência da substring ou -1 se não encontrado.
    • string.lastIndexOf(substring): Retorna o índice da última ocorrência da substring ou -1 se não encontrado.
    • string.slice(start, end): Retorna uma parte da string entre os índices especificados.
    • string.split(delimiter): Divide a string em um array de substrings com base no delimitador.
    • string.trim(): Remove espaços em branco do início e do fim da string.
    • string.toLowerCase(): Converte todos os caracteres da string para letras minúsculas.
    • string.toUpperCase(): Converte todos os caracteres da string para letras maiúsculas.
    • string.repeat(count): Retorna uma nova string repetindo a string original um número especificado de vezes.
    • string.padStart(targetLength, padString): Preenche a string com outra string até atingir o comprimento desejado a partir do início.
    • string.padEnd(targetLength, padString): Preenche a string com outra string até atingir o comprimento desejado a partir do final.
    • string.localeCompare(compareString): Compara duas strings em ordem local.
    • string.replace(searchValue, newValue): Substitui parte da string por outra substring.
  • Boolean: Representa um valor verdadeiro (true) ou falso (false).
  • Object: Estrutura de dados que armazena pares chave-valor.
  • Array: Tipo de objeto usado para armazenar listas de valores.
  • Function: Um objeto que pode ser chamado (invocado).
  • Undefined: Indica que uma variável não possui valor atribuído.
  • Null: Representa a ausência intencional de qualquer valor.
  • Symbol: Representa um identificador único e imutável.
  • BigInt: Representa números inteiros muito grandes.

2. Objetos Globais

  • Math: Objeto que contém propriedades e métodos para constantes e funções matemáticas.
    • Math.random(): Retorna um número pseudoaleatório entre 0 e 1.
    • Math.abs(x): Retorna o valor absoluto de x.
    • Math.ceil(x): Arredonda x para cima.
    • Math.floor(x): Arredonda x para baixo.
    • Math.round(x): Arredonda x para o inteiro mais próximo.
    • Math.sqrt(x): Retorna a raiz quadrada de x.
    • Math.pow(base, exponent): Retorna base elevado à potência de exponent.
    • Math.log(x): Retorna o logaritmo natural de x.
    • Math.log10(x): Retorna o logaritmo base 10 de x.
    • Math.exp(x): Retorna o valor de e elevado à potência de x.
    • Math.hypot(...values): Retorna a raiz quadrada da soma dos quadrados de seus argumentos.
  • Infinity: Representa um valor infinito.
  • NaN: Representa um valor que não é um número (Not-a-Number).
  • undefined: Tipo que indica que uma variável não possui valor.

3. Manipulação de Arrays

  • Array.isArray(value): Verifica se value é um array.
  • array.push(element): Adiciona um ou mais elementos ao final do array e retorna o novo comprimento do array.
  • array.pop(): Remove e retorna o último elemento do array.
  • array.shift(): Remove e retorna o primeiro elemento do array.
  • array.unshift(element): Adiciona um ou mais elementos ao início do array.
  • array.concat(...arrays): Une dois ou mais arrays.
  • array.slice(start, end): Retorna uma cópia de uma parte do array entre os índices especificados.
  • array.splice(start, deleteCount, ...items): Altera o conteúdo de um array, adicionando ou removendo elementos.
  • array.fill(value, start, end): Preenche todos os elementos do array com um valor estático.
  • array.reverse(): Inverte a ordem dos elementos do array.
  • array.sort(compareFunction): Classifica os elementos do array.
  • array.map(callback): Cria um novo array com os resultados da chamada de callback em cada elemento do array.
  • array.filter(callback): Cria um novo array com todos os elementos que passam no teste implementado pela função fornecida.
  • array.reduce(callback, initialValue): Aplica uma função a um acumulador e a cada elemento do array para reduzi-lo a um único valor.
  • array.find(callback): Retorna o primeiro elemento que satisfaz a função de teste.
  • array.every(callback): Verifica se todos os elementos do array passam no teste implementado pela função fornecida.
  • array.some(callback): Verifica se pelo menos um dos elementos do array passa no teste implementado pela função fornecida.
  • array.includes(value): Verifica se um array contém um determinado valor.
  • array.indexOf(value): Retorna o índice da primeira ocorrência de um elemento no array.
  • array.findIndex(callback): Retorna o índice do primeiro elemento que satisfaz a função de teste.
  • array.join(separator): Junta todos os elementos de um array em uma string, usando o separador especificado.
  • array.keys(): Retorna um novo objeto Array Iterator com as chaves de cada índice do array.
  • array.values(): Retorna um novo objeto Array Iterator com os valores de cada índice do array.
  • array.entries(): Retorna um novo objeto Array Iterator com os pares chave/valor de cada índice do array.

4. Manipulação de Objetos

  • Object.keys(obj): Retorna um array com os nomes das propriedades enumeráveis de um objeto.
  • Object.values(obj): Retorna um array com os valores das propriedades enumeráveis de um objeto.
  • Object.entries(obj): Retorna um array de pares [chave, valor] de um objeto.
  • Object.assign(target, ...sources): Copia valores de todas as propriedades de um ou mais objetos de origem para um objeto de destino.
  • Object.create(proto): Cria um novo objeto, usando um objeto existente como protótipo.
  • Object.freeze(obj): Congela um objeto, impedindo modificações.
  • Object.seal(obj): Impede novas adições e elimina a exclusão de propriedades.
  • Object.defineProperty(obj, prop, descriptor): Define uma nova propriedade em um objeto ou modifica uma existente.
  • Object.defineProperties(obj, props): Define várias propriedades em um objeto.
  • Object.getOwnPropertyNames(obj): Retorna um array com os nomes das propriedades de um objeto, incluindo não enumeráveis.
  • Object.getOwnPropertyDescriptor(obj, prop): Retorna um descritor de propriedade de um objeto.
  • Object.hasOwnProperty(prop): Verifica se um objeto possui uma propriedade específica como sua própria propriedade (não herdada).

5. Manipulação de Funções

  • function() {}: Define uma função.
  • return: Retorna um valor de uma função.
  • Function.prototype.call(thisArg, ...args): Chama uma função com um dado valor this e argumentos fornecidos individualmente.
  • Function.prototype.apply(thisArg, argsArray): Chama uma função com um dado valor this e um array de argumentos.
  • Function.prototype.bind(thisArg, ...args): Cria uma nova função que, quando chamada, tem seu this definido para um valor específico.
  • Function.prototype.toString(): Retorna uma string representando o código-fonte da função.

6. Manipulação de Erros

  • try...catch: Estrutura para tratamento de exceções.
  • throw: Lança uma exceção.
  • TypeError: Lançado quando uma variável não é do tipo esperado.
  • ReferenceError: Lançado quando uma referência não pode ser encontrada.
  • SyntaxError: Lançado quando há um erro de sintaxe no código.

7. Manipulação de Tempo

  • setTimeout(callback, delay): Executa uma função após um determinado tempo.
  • setInterval(callback, delay): Executa uma função repetidamente, com um intervalo de tempo especificado.
  • clearTimeout(timeoutId): Cancela um timer definido por setTimeout().
  • clearInterval(intervalId): Cancela um timer definido por setInterval().

8. Manipulação de Dados Estruturados

  • Map
    • new Map(): Cria um novo mapa.
    • map.set(key, value): Adiciona ou atualiza um par chave-valor.
    • map.get(key): Retorna o valor associado a uma chave.
    • map.delete(key): Remove um par chave-valor.
    • map.has(key): Verifica se uma chave existe no mapa.
    • map.forEach(callback): Executa uma função para cada par chave-valor no mapa.
    • map.size: Retorna o número de pares chave-valor no mapa.
  • WeakMap
    • new WeakMap(): Cria um novo WeakMap.
    • weakMap.set(key, value): Adiciona ou atualiza um par chave-valor.
    • weakMap.get(key): Retorna o valor associado a uma chave.
    • weakMap.delete(key): Remove um par chave-valor.
    • weakMap.has(key): Verifica se uma chave existe no WeakMap.
  • Set
    • new Set(): Cria um novo conjunto.
    • set.add(value): Adiciona um valor ao conjunto.
    • set.delete(value): Remove um valor do conjunto.
    • set.has(value): Verifica se um valor está presente no conjunto.
    • set.forEach(callback): Executa uma função para cada valor do conjunto.
    • set.size: Retorna o número de valores únicos no conjunto.
  • WeakSet
    • new WeakSet(): Cria um novo WeakSet.
    • weakSet.add(value): Adiciona um valor ao WeakSet.
    • weakSet.delete(value): Remove um valor do WeakSet.
    • weakSet.has(value): Verifica se um valor está presente no WeakSet.

9. Iteradores e Geradores

  • Generator Functions
    • function*: Declara uma função geradora que pode ser pausada e retomada.
    • yield: Pausa a execução da função geradora e retorna um valor.
    • Symbol.iterator: Propriedade que retorna o método do iterador de um objeto.

10. Manipulação de Promessas

  • Promise
    • Promise.all(iterable): Retorna uma promessa que resolve quando todas as promessas fornecidas são resolvidas.
    • Promise.race(iterable): Retorna uma promessa que resolve ou rejeita assim que uma das promessas fornecidas resolve ou rejeita.
    • Promise.allSettled(iterable): Retorna uma promessa que resolve após todas as promessas fornecidas terem sido concluídas, independentemente de serem resolvidas ou rejeitadas.
    • Promise.any(iterable): Retorna uma promessa que resolve assim que uma das promessas fornecidas é resolvida, ou rejeita se todas forem rejeitadas.
    • Promise.resolve(value): Retorna uma promessa que é resolvida com um valor.
    • Promise.reject(reason): Retorna uma promessa que é rejeitada com um motivo.

11. APIs de Web

  • Fetch API
    • fetch(url): Faz uma requisição de rede e retorna uma promessa.
    • Response.json(): Retorna uma promessa que resolve com o corpo da resposta como JSON.
    • Response.ok: Indica se a resposta foi bem-sucedida (código HTTP no intervalo 200-299).
    • Response.status: Retorna o código de status HTTP da resposta.
  • LocalStorage
    • localStorage.setItem(key, value): Armazena um item na local storage.
    • localStorage.getItem(key): Recupera um item da local storage.
  • SessionStorage
    • sessionStorage.setItem(key, value): Armazena um item na session storage.
    • sessionStorage.getItem(key): Recupera um item da session storage.

12. Manipulação de Eventos

  • Eventos
    • addEventListener(type, listener): Adiciona um ouvinte de evento a um elemento.
    • removeEventListener(type, listener): Remove um ouvinte de evento de um elemento.
    • event.preventDefault(): Impede o comportamento padrão de um evento.
    • event.stopPropagation(): Impede que o evento se propague para outros elementos.
    • event.stopImmediatePropagation(): Impede que outros manipuladores do mesmo evento sejam chamados.
  • CustomEvent: Permite criar eventos personalizados.
    • new CustomEvent(type, options): Cria um novo evento personalizado.

13. Manipulação do DOM

  • document.getElementById(id): Retorna o elemento com o ID especificado.
  • document.querySelector(selector): Retorna o primeiro elemento que corresponde ao seletor CSS fornecido.
  • document.createElement(tagName): Cria um novo elemento HTML.
  • document.getElementsByClassName(className): Retorna uma coleção de elementos que possuem a classe especificada.
  • document.getElementsByTagName(tagName): Retorna uma coleção de elementos que correspondem ao nome da tag especificada.
  • element.classList: Propriedade que retorna a lista de classes de um elemento.
  • element.setAttribute(name, value): Define um atributo em um elemento.
  • element.appendChild(child): Adiciona um novo filho a um elemento.
  • element.remove(): Remove um elemento do DOM.

14. Modules

  • import { name } from './module.js': Importa um membro nomeado de um módulo.
  • export const name = value;: Exporta uma variável de um módulo.

15. Templates Literais

  • ${expression}: Permite interpolação de expressões dentro de strings.

16. Symbol

  • Symbol.for(key): Cria ou recupera um símbolo global.

17. Desestruturação

  • Desestruturação de Arrays: Permite desempacotar valores de arrays em variáveis distintas.
  • Desestruturação de Objetos: Permite extrair valores de objetos em variáveis distintas.
  • Desestruturação aninhada: Permite extrair valores de arrays ou objetos aninhados.

18. Proxy e Reflect

  • Proxy
    • new Proxy(target, handler): Permite criar um objeto que pode interceptar operações em outro objeto.
  • Reflect
    • Reflect.get(target, propertyKey): Obtém a propriedade de um objeto.
    • Reflect.set(target, propertyKey, value): Define a propriedade de um objeto.

19. API de Geolocalização

  • navigator.geolocation.getCurrentPosition(success, error, options): Obtém a posição atual do usuário.

20. Eventos de Teclado e Mouse

  • Eventos de Teclado
    • keydown: Disparado quando uma tecla é pressionada.
    • keyup: Disparado quando uma tecla é solta.
    • keypress: Disparado quando uma tecla é pressionada (deprecado em algumas situações).
  • Eventos de Mouse
    • click: Disparado quando um elemento é clicado.
    • dblclick: Disparado quando um elemento é clicado duas vezes.
    • mouseover: Disparado quando o ponteiro do mouse entra em um elemento.
    • mouseout: Disparado quando o ponteiro do mouse sai de um elemento.
