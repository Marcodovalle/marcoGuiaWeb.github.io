Guia de Propriedades e Funções do CSS

1. Layout

  • display: Controla como um elemento é exibido (ex.: block, inline, flex, grid, none).
  • position: Controla o posicionamento de um elemento (static, relative, absolute, fixed, sticky).
  • top, right, bottom, left: Especifica a posição de elementos posicionados.
  • z-index: Controla a ordem de empilhamento de elementos.
  • float: Faz um elemento flutuar à esquerda ou à direita.
  • clear: Controla o comportamento de flutuação de elementos adjacentes.

2. Tamanho

  • width: Define a largura de um elemento.
  • height: Define a altura de um elemento.
  • max-width: Define a largura máxima de um elemento.
  • max-height: Define a altura máxima de um elemento.
  • min-width: Define a largura mínima de um elemento.
  • min-height: Define a altura mínima de um elemento.
  • box-sizing: Controla como a largura e altura de um elemento são calculadas.

3. Espaçamento

  • margin: Define o espaço externo ao redor de um elemento.
  • padding: Define o espaço interno entre o conteúdo e a borda do elemento.
  • border: Define a borda de um elemento.

4. Tipografia

  • font-family: Define a fonte do texto.
  • font-size: Define o tamanho do texto.
  • font-weight: Define a espessura do texto.
  • font-style: Define o estilo da fonte.
  • line-height: Define a altura da linha.
  • text-align: Alinha o texto.
  • color: Define a cor do texto.
  • text-transform: Controla a transformação do texto.
  • letter-spacing: Controla o espaçamento entre letras.
  • word-spacing: Controla o espaçamento entre palavras.
  • text-decoration: Adiciona decorações ao texto.
  • white-space: Controla como o espaço em branco é tratado.

5. Cores e Fundos

  • background-color: Define a cor de fundo de um elemento.
  • background-image: Define uma imagem de fundo.
  • background-repeat: Controla como a imagem de fundo se repete.
  • background-position: Define a posição da imagem de fundo.
  • opacity: Define a opacidade de um elemento.

6. Bordas e Sombreamento

  • border-radius: Arredonda os cantos de um elemento.
  • border-style: Define o estilo da borda.
  • border-width: Define a largura da borda.
  • box-shadow: Adiciona uma sombra ao redor de um elemento.
  • text-shadow: Adiciona uma sombra ao texto.

7. Transformações e Transições

  • transform: Aplica transformações a um elemento.
  • transition: Define a transição entre estados de um elemento.
  • animation: Define animações CSS.

8. Flexbox

  • flex-direction: Define a direção dos itens em um contêiner flexível.
  • flex-wrap: Controla se os itens devem ou não quebrar a linha.
  • justify-content: Alinha os itens ao longo do eixo principal.
  • align-items: Alinha os itens ao longo do eixo transversal.
  • align-content: Alinha as linhas de itens em um contêiner flexível.

9. Grid

  • grid-template-columns: Define as colunas em um contêiner de grade.
  • grid-template-rows: Define as linhas em um contêiner de grade.
  • grid-area: Define o espaço de um item em uma grade.
  • grid-gap: Define o espaço entre as linhas e colunas da grade.

Funções CSS

1. Funções de Cores

  • rgb(): Define uma cor usando valores de vermelho, verde e azul.
  • rgba(): Define uma cor com um valor alfa para opacidade.
  • hsl(): Define uma cor usando matiz, saturação e luminosidade.
  • hsla(): Como hsl(), mas com um valor alfa.

2. Funções de Medida

  • calc(): Realiza cálculos para determinar valores CSS.
  • var(): Acessa variáveis CSS definidas com --.

3. Funções de Gradiente

  • linear-gradient(): Cria um gradiente linear.
  • radial-gradient(): Cria um gradiente radial.
  • conic-gradient(): Cria um gradiente cônico.

Media Queries

  • @media: Permite aplicar estilos diferentes com base em condições (ex.: tamanho da tela, orientação).

Pseudo-classes e Pseudo-elementos

  • :hover: Aplica estilo quando o mouse está sobre um elemento.
  • :focus: Aplica estilo quando um elemento recebe foco.
  • :active: Aplica estilo quando um elemento está sendo ativado.
  • :visited: Aplica estilo a links visitados.
  • :first-child: Aplica estilo ao primeiro filho de um elemento.
  • :last-child: Aplica estilo ao último filho de um elemento.
  • :nth-child(n): Aplica estilo ao n-ésimo filho de um elemento.
  • :nth-of-type(n): Aplica estilo ao n-ésimo filho de um tipo específico.
  • ::before: Adiciona conteúdo antes de um elemento.
  • ::after: Adiciona conteúdo depois de um elemento.
  • ::placeholder: Estiliza o texto do placeholder em campos de entrada.
