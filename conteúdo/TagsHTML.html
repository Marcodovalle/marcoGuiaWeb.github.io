Guia de Tags HTML5 e Suas Funções

Estrutura Básica

  • <!DOCTYPE html>: Declaração que define o documento como HTML5.
  • <html>: Tag raiz que contém todos os outros elementos do documento.
  • <head>: Contém metadados sobre o documento (título, links, scripts).
  • <body>: Contém o conteúdo visível da página.

Metadados e Informações do Documento

  • <meta>: Define metadados como charset, descrição, viewport, etc.
  • <title>: Define o título da página (aparece na aba do navegador).
  • <link>: Usado para vincular folhas de estilo externas ou outros recursos.
  • <style>: Contém definições CSS diretamente no documento.
  • <base>: Define a URL base para todos os links relativos no documento.

Conteúdo de Cabeçalho

  • <header>: Define um cabeçalho de página ou de uma seção.
  • <nav>: Define uma área de navegação, com links para outras partes do site.
  • <h1> ... <h6>: Define títulos de diferentes níveis de importância (de 1 a 6).
  • <section>: Define uma seção ou parte de um documento.
  • <article>: Define um conteúdo independente, como uma notícia ou um post.
  • <aside>: Define conteúdo relacionado, como uma barra lateral.
  • <footer>: Define o rodapé da página ou de uma seção.

Agrupamento de Conteúdo

  • <p>: Define um parágrafo.
  • <div>: Define uma divisão ou seção genérica no documento.
  • <span>: Define um contêiner em linha, geralmente usado para aplicar estilos.

Listas

  • <ul>: Define uma lista não ordenada.
  • <ol>: Define uma lista ordenada.
  • <li>: Define um item em uma lista.
  • <dl>: Define uma lista de descrição.
  • <dt>: Define um termo em uma lista de descrição.
  • <dd>: Define a descrição de um termo em uma lista de descrição.

Links e Navegação

  • <a>: Define um link (âncora).
  • <area>: Define uma área clicável dentro de um mapa de imagem.
  • <nav>: Define um bloco de navegação.

Multimídia

  • <img>: Define uma imagem.
  • <figure>: Define uma figura, geralmente com uma legenda.
  • <figcaption>: Define a legenda para uma figura.
  • <audio>: Define conteúdo de áudio.
  • <video>: Define conteúdo de vídeo.
  • <source>: Define múltiplos recursos de mídia para <audio> ou <video>.
  • <track>: Define faixas de texto para <video> ou <audio>, como legendas.
  • <embed>: Define um contêiner para recursos externos (por exemplo, um vídeo do YouTube).
  • <object>: Define um contêiner para conteúdo embutido.
  • <param>: Define parâmetros para um <object>.

Tabelas

  • <table>: Define uma tabela.
  • <caption>: Define uma legenda para uma tabela.
  • <thead>: Agrupa o cabeçalho da tabela.
  • <tbody>: Agrupa o corpo da tabela.
  • <tfoot>: Agrupa o rodapé da tabela.
  • <tr>: Define uma linha de tabela.
  • <td>: Define uma célula de dados de tabela.
  • <th>: Define uma célula de cabeçalho de tabela.

Formulários

  • <form>: Define um formulário para entrada de dados do usuário.
  • <input>: Define um campo de entrada de dados.
  • <textarea>: Define uma área de texto (múltiplas linhas).
  • <button>: Define um botão clicável.
  • <select>: Define uma lista suspensa.
  • <option>: Define uma opção dentro de um <select>.
  • <label>: Define um rótulo para um elemento de formulário.
  • <fieldset>: Agrupa campos de um formulário.
  • <legend>: Define uma legenda para um <fieldset>.
  • <datalist>: Define uma lista de opções predefinidas para um campo de entrada.
  • <output>: Exibe o resultado de um cálculo.
  • <progress>: Representa o progresso de uma tarefa.
  • <meter>: Representa uma medição ou valor dentro de um intervalo conhecido.

Semântica e Organização

  • <main>: Define o conteúdo principal de um documento.
  • <mark>: Destaca texto.
  • <time>: Define uma data ou hora.
  • <address>: Define informações de contato.
  • <blockquote>: Define uma citação em bloco.
  • <cite>: Define o título de uma obra citada.
  • <q>: Define uma citação curta.
  • <code>: Define um trecho de código.
  • <pre>: Define texto pré-formatado.
  • <kbd>: Define entrada do teclado.
  • <samp>: Define uma saída de código (ou de sistema).
  • <var>: Define uma variável.
  • <b>: Define texto em negrito sem ênfase semântica.
  • <strong>: Define texto com ênfase forte.
  • <i>: Define texto em itálico sem ênfase semântica.
  • <em>: Define texto com ênfase.

Scripts e Interatividade

  • <script>: Define scripts (geralmente JavaScript).
  • <noscript>: Define conteúdo alternativo para navegadores que não suportam scripts.
  • <canvas>: Define uma área de desenho em gráficos 2D.
  • <svg>: Define gráficos vetoriais escaláveis.
  • <template>: Define um template reutilizável.
